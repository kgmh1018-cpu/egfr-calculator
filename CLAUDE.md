# CLAUDE.md

## 관리 규칙
- 현재 상태만, 과정/날짜 금지
- 함수는 시그니처 + 한 줄 설명만
- 코드 보면 아는 건 빼기
- 100줄 이내 유지
- 업데이트 요청 시 추가/수정/삭제 내용만 뽑기
- 파일 수정 시 전체 재출력 금지, 찾기/바꾸기 형식만 사용

## 나에 대해
- 본 파일은 내가 명시적으로 말하기 전엔 건드리지 마
- 파일 300줄 초과, 구조 문제, 잠재적 버그 보이면 먼저 경고
- 더 나은 구현 방법이나 개선점 보이면 먼저 제안해줘
- 내 요청보다 더 좋은 방향이 있으면 이유와 함께 밀어줘

## 프로젝트 구조
CKD-EPI 2021 기반 eGFR 계산기 + CKD 단계별 약물 위험 안내 웹앱
- 순수 바닐라 JS / CSS / HTML. 빌드 없음. 외부 의존성 없음.
- `index.html` — 레이아웃
- `style.css` — 스타일 (CSS 변수 기반 다크 테마)
- `app.js` — 계산 로직, UI 렌더링, 이벤트 핸들링
- `data.js` — STAGES / DRUGS / CATS 배열 (순수 데이터만)

## 도메인 지식
- CKD-EPI 2021 공식으로 sCr → eGFR 계산
- DRUGS 각 항목: `ci_at`(금기 GFR), `ca_at`(주의 GFR), `imp`(중요 여부)
- eGFR 기준 약물 상태: `'safe'` / `'ca'` / `'ci'` (getRisk 반환값)
- 슬라이더 조작은 syncFromSCR / syncFromGFR / syncFromAge / syncFromStage 통해서만
- state.updating 플래그 직접 건드리지 말 것 (무한루프 방지용)
- 새 렌더 로직은 render() 안에 추가
